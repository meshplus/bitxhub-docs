# 基本跨链操作

以一个基本的以太坊和fabric之间的跨链为例，其中bitxhub和pier均以二进制方式启动。

在进行跨链操作前，需要先安装好goduck并完成初始化
```shell
$ cd $HOME && git clone https://github.com/meshplus/goduck.git
$ cd $HOME/goduck && git checkout release-1.23
$ git pull
$ make install && goduck init
```
